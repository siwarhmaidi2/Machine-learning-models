In the highly competitive telecom industry, retaining customers is crucial, and identifying factors that contribute to customer churn can help in implementing targeted retention strategies. Machine learning methods are valuable tools in predicting churn and providing insights into customer behavior.
⚙️ Techniques and Tools Used:
- Data Preprocessing: Addressed imbalanced data, feature engineering, and normalization.
- Models: Logistic Regression , Random Forest , Support Vector Classifier (SVC) , Gradient Boosting (GBM) , decisionTree
